Dat was het verkooppraatje over Zermos. Ik heb dit project over een aantal jaren gemaakt en dat heb ik met veel plezier gedaan. Het is uiteindelijk zover gekomen dat ik posters door mijn school had gehangen om meer mede-leerlingen te overtuigen het ook te gebruiken, wat uiteindelijk maar +/- 100 gebruikers opleverde, maar het was wel ultiem gaaf natuurlijk!

Nu ben ik klaar met de middelbare school en ik vind dat dit project nu voor iedereen bereikbaar moet zijn, vanwege de vele tijd die gegaan is in de koppelingen, het uitvogelen van veel aspecten van Somtoday en Infowijs. Nu deel ik dit werk met jullie allemaal, leer ervan, bouw erop verder, doe wat je wil! (conform licientsie aub).

Dit is wel een archief van alle code geschreven voor dit project, het is dus niet direct bedoeld om voor iedereen om een eigen instance te gaan hosten, het mag zeker, maar het project is volledig gebouwd naar mijn eigen infrastructuur! Ook wil ik meegeven dat ik heel veel fouten heb gemaakt, zoals voornamelijk het hardcoden van wachtowwrden voor mijn  database die nu wel allemaal gepatcht zijn, dus geloof me, als je denkt een wachtrwoord te vinden, dan is dat al lang weer verholpen en een oud wachtwoord :)
